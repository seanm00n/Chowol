# Chowol
- "Chowol"은 MMORPG 게임 "로스트 아크"의 초월 시스템을 유니티로 구현한 프로젝트입니다.
- 이 프로젝트는 강화 시스템의 작동 방식을 이해하기 위해 만들었습니다.

# 주요 기능
- 플레이어는 원하는 장비 부위와 초월 단계를 선택하여 초월할 수 있으며, 사용된 재화를 확인할 수 있습니다.
